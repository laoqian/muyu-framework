#  muyu快速开发框架
    利用spring-boot构建后端，redis作为系统缓存，react构建前端，webpack为前端发布工具的项目管理系统。目前开放时前后端通信方式，react->node server->proxy middleware->springboot
###1、sprinboot启动
>server目录存放的springboot程序代码。按照一般maven项目启动即可。

###2、client server启动
>cd  client && npm start
